WHAT ARE HTML TABLES?

  • HTML tables are used and they allow web authors to arrange data like text,images, links, other tables, etc. into rows and columns of cells.

  • A table is defined using the 'table' element and contains a number of table cells (td-->for table data)

  • (tr) tag is used to create table rows.

  • Table cells can be merged using the colspan and the rowspan attributes

  • A caption can be added to a table using the 'caption' element.
