Linking Documents:-

A link is specified using HTML tag <a>. This tag is called anchor tag and anything between the opening <a> tag and the closing </a> tag becomes part of the link and a user can click that part to reach to the linked document.
Following is the simple syntax to use <a> tag:   <a href = "Document URL" ... attributes-list>Link Text</a>

Example of linking document

Use of Base Path :-

When we link HTML documents related to the same website, it is not required to give a complete URL for every link. We can get rid of it if we use <base> tag in our HTML document header. This tag is used to give a base path for all the links. So our browser will concatenate given relative path to this base path and will make a complete URL.
